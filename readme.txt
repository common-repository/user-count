== Description ==

Simply displays the number of users who have an account on your blog to the admin dashboard. Also lists number of users by user role.


== Installation ==

1. Upload the usercount folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
